# 更新日志

## 1.0.14 - 2026-06-04

- 新增沙盒管理后端接口 `/api/sandbox/environments`、`/api/sandbox/run` 与 `/api/sandbox/records`，支持沙盒环境维护、JS 片段运行和运行记录查询。
- 沙盒环境支持配置允许目录、命令 allowlist、执行超时和输出上限，默认环境限制在 `plugins/example` 与沙盒数据目录。
- 新增沙盒管理页面，提供环境列表、运行控制台、最近运行结果和运行记录表。
- 沙盒环境创建、编辑、删除、启停与代码运行写入审计日志。
- 同步前端版本号与构建产物，面板版本展示更新为 `1.0.14`。

## 1.0.13 - 2026-06-04

- 修正 JS 插件管理范围，仅扫描和操作 `plugins/example` 目录内的 `.js`、`.js.disabled` 与 `.js.exa` 文件。
- JS 插件管理页面标题与路径展示调整为 `plugins/example`，`plugins/other`、`plugins/system` 与 `plugins/adapter` 保持系统插件和适配器目录边界。
- 同步前端版本号与构建产物，面板版本展示更新为 `1.0.13`。

## 1.0.12 - 2026-06-04

- 新增 JS 插件管理后端接口 `/api/js-plugins/list`、`/api/js-plugins/detail`、`/api/js-plugins/toggle`、`/api/js-plugins/copy-example` 与 `/api/js-plugins/reload`。
- 新增 JS 插件管理页面，支持按状态和关键词查看 `plugins` 目录内 JS 插件文件、规则、任务与源码。
- JS 插件管理支持启用、禁用、复制 `.js.exa` 示例文件与热重载已启用插件文件，相关操作写入审计日志。
- 同步前端版本号与构建产物，面板版本展示更新为 `1.0.12`。

## 1.0.11 - 2026-06-04

- 控制台新增 stdin 命令发送接口 `/api/console/input`，前端可直接向 Yunzai 标准输入发送单行命令。
- 实时控制台页面新增命令输入栏，发送后通过现有 stdin adapter 进入云崽命令处理链路。
- 修复服务进程未挂载 TTY 时控制台命令误报“标准输入未连接”的问题，改为通过 stdin adapter 投递命令。
- 控制台日志行展示精简为“序号 + 云崽原始日志内容”，移除页面上的时间、等级和来源列。
- 实时控制台支持渲染 ANSI 颜色，保留 Yunzai 自定义控制台配色效果。
- 控制台命令发送接入审计日志，仅记录命令长度和执行结果。
- 同步前端版本号与构建产物，面板版本展示更新为 `1.0.11`。

## 1.0.10 - 2026-06-04

- 控制台新增实时流接口 `/api/console/stream`，通过 SSE 推送当前进程 `stdout/stderr` 终端输出。
- 控制台页面改为实时终端模式，支持连接、断开、清屏和自动滚动。
- 保留 `/api/console/logs` 作为历史日志回看接口，主控制台入口默认使用实时流。
- 同步前端版本号与构建产物，面板版本展示更新为 `1.0.10`。

## 1.0.9 - 2026-06-04

- 新增控制台后端接口 `/api/console/logs`，支持读取 Yunzai 命令日志和错误日志尾部内容。
- 新增控制台页面，支持命令日志日期切换、关键词筛选、行数限制和手动刷新。
- 控制台日志读取会去除 ANSI 控制字符，并在大日志文件中只读取尾部内容以降低面板开销。
- 同步前端版本号与构建产物，面板版本展示更新为 `1.0.9`。

## 1.0.8 - 2026-06-04

- 任务管理新增插件任务执行记录接口 `/api/tasks/plugin/records`，解析 Yunzai `command` 日志中的 `[TASK]` 运行记录。
- 任务管理页面调整为“插件任务 / 任务执行记录 / 后台执行记录”三标签，插件任务执行记录展示插件、任务名、Cron、状态和耗时。
- 保留代发消息等锅巴后台执行记录入口，并同步前端版本号与构建产物到 `1.0.8`。

## 1.0.7 - 2026-06-04

- 任务管理新增插件定时任务接口 `/api/tasks/plugin`，读取 Yunzai 运行时已注册的 `plugin.task` 任务。
- 任务管理页面改为“插件任务 / 执行记录”双标签，插件任务展示插件、任务名、Cron、下次运行、方法、日志和注册状态。
- 保留代发消息等后台执行记录，继续支持状态、类型和关键词筛选。
- 同步前端版本号与构建产物，面板版本展示更新为 `1.0.7`。

## 1.0.6 - 2026-06-04

- 新增文件管理后端接口 `/api/files`，支持云崽根目录内的快捷根目录、目录列表、文件信息、文本读取与保存。
- 新增双栏文件管理页面，支持位置栏、路径跳转、点击文件打开、新建目录、重命名、删除文件或空目录、在线编辑 1MB 内文本文件。
- 文件写入、创建目录、重命名和删除操作接入审计日志，记录到 `data/guoba/audit`。
- 文件管理路径统一限制在云崽根目录内，删除操作当前限定为文件或空目录。
- 同步前端版本号与构建产物，面板版本展示更新为 `1.0.6`。

## 1.0.5 - 2026-06-04

- 新增代发消息后端接口 `/api/messages/send`，支持选择账号向私聊或群/频道发送纯文本消息。
- 新增任务管理基础接口 `/api/tasks`，任务记录短期持久化到 `data/guoba/tasks/recent.json`，支持分页、状态、类型和关键词查询。
- 新增代发消息页面，支持选择发送账号、切换私聊/群聊、搜索好友或群/频道并查看最近发送任务。
- 新增任务管理页面，支持按任务状态、任务类型和关键词筛选任务记录。
- 新增消息发送审计记录，发送成功或失败都会写入 `data/guoba/audit` 并关联任务 ID。
- 同步前端版本号与构建产物，面板版本展示更新为 `1.0.5`。

## 1.0.4 - 2026-06-04

- 同步 Express 5 兼容改造，移除 `body-parser` 依赖，升级 `express`、`multer` 与 `jsonwebtoken`。
- 修复共享 Yunzai / TRSS 端口时请求体被重复读取导致登录、退出接口报 `stream is not readable` 的问题。
- 修复 Express 5 下通配路由注册，上传中间件与 404 兜底路由改为命名通配写法。
- 新增账号管理基础接口 `/api/accounts`，支持账号分页查询、详情、诊断、好友列表、群/频道列表与账号上下线操作。
- 优化账号管理页面，新增搜索、分页、详情抽屉、账号能力诊断、好友与群/频道查看。
- 新增操作审计日志基础能力，账号状态变更会写入 `data/guoba/audit`。
- 保留旧版 `/user/list` 与 `/user/account/status` 兼容入口，前端账号状态操作切换到新版账号接口。
- 调整账号管理字段范围，移除账号备注、标签和默认账号相关入口。
- 同步前端版本号与构建产物，面板版本展示更新为 `1.0.4`。

## 1.0.3 - 2026-05-04

- 重写群组配置页布局，基础设置按场景分区展示，功能名单区域支持按插件一键加入、补全、移入或移出功能规则。
- 群组功能白名单 / 黑名单继续保存为 Yunzai 原生 `enable` / `disable` 字符串数组，批量加入时会自动处理另一侧名单中的同名功能，避免配置冲突。
- 优化新增群配置弹窗，支持完整拉取 Bot 会话列表，排除标准输入，并按 QQ 群、官方 QQ 机器人、Discord、其他会话类型筛选。
- Discord 会话按“服务器 / 频道”层级展示，官方 QQ 机器人会话独立标识，长 ID 列和弹窗尺寸同步加宽。
- 优化配置管理二级菜单图标，基础配置、群组配置、原神配置、其他配置不再共用齿轮图标。

## 1.0.2 - 2026-05-02

- 配置管理改为侧边栏二级子页面，移除页面内 Tab，基础配置、群组配置、原神配置、其他配置可直接从菜单进入。
- 兼容私库 `plugins/genshin` 配置路径，原神配置优先读取 `plugins/genshin/config/config/*.yaml`，旧路径继续兜底。
- 修复功能规则按插件菜单过滤时，`pluginInfo.name` 与 `package.json name` 大小写或来源不一致导致规则显示为空的问题。
- 新增 HTTPS/SSL 监听配置，可直接使用证书文件启动锅巴 HTTPS 服务。
- 新增 GitHub 代理全局工具，安装插件、读取 README、插件索引等 GitHub 链路统一复用代理配置。
- 优化插件安装体验，支持选择包管理器、跳过依赖安装、自动重启开关，并在前端展示安装日志。
- 新增固定密码登录，设置密码后可在验证码登录与密码登录之间任选其一；密码使用 PBKDF2-SHA256 哈希保存，设置/修改需后台验证码确认。
- 新增面板资料设置，支持右上角头像菜单修改昵称、上传并裁剪自定义头像，移除默认示例邮箱与 Pro 标签。
- 新增功能规则只读查看器，基于运行时规则按插件 package 名、JS 文件、功能实例和规则分层展示，支持搜索、筛选、规则详情查看。
- 调整插件管理导航，插件市场独立成页，功能规则改为侧栏二级菜单，规则页内容区移除重复插件列表。
- 清理旧天气、旧首页随机图片、旧自动更新检查与旧资源仓库初始化逻辑。
- 整理维护目录，新增项目结构说明与 Codex 协作说明。
